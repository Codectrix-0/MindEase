# 📌 About The Project

**MindEase** is a beginner-friendly web project created to provide a calming and organized digital experience.  
The project focuses on clean UI design, interactive functionality, and improving frontend development skills.

This project was developed as part of my learning journey in **Web Development** and helped me understand how websites work using HTML, CSS, and JavaScript.
